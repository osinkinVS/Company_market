# Сбор данных о российских BTL и маркетинговых агентств
Проект содержит файлы и скрипты для сбора данных из открытых источников
## Источники данных: 
1. Рейтинг РРАР-2025 - базовый список агентств
2. Публичные рейтинги (marketing-tech.ru)
3. Базы данных (Dadata, Rusprofile, Checko, RUWARD)
4. Сайты компаний
## Структура проекта
1. Company.xlxs - сырые данные, собранные из источников (1, 2). Названия компаний, сфера деятельности, выручка
2. find_inn.py - сбор по API со справочника DaData.ru, получение по названию компании её ИНН, основной ОКВЭД и наименование Юр.лица
3. companies_1.csv - автоматически собранная информация из find_inn.py
4. all_on_inn.py - по полученным ИНН сбор остальной информации с Checko по API (итог checko_contacts.csv). Получил номера телефонов, email, адрес сайта и с\с количество сотрудников
5. dataframe.xlsx и dataframe.csv - результат обработки полученной информации, слияние по инн файлов companies_1.csv и checko_contacts.csv, обработка значений, типов данных и сохранение в общий конечный файл с разными форматами
6. final.xlsx - результат ручной обработки и проверки достоверности, также добавление недостающей информации с сайтов компаний и сервисов (DaData, Rusprofile, RUWARD). Конечный файл
7. ***companies.csv*** - финальный результат в формате csv
## Подход
Удалось реализовать:
- Автоматизированный сбор базовых данных
- Экспорт структурированных данных в CSV
- Слияние данных по инн из разных источников

Не успели, но планировали:
- Систему верификации через анализ реальных кейсов агентств
- Мониторинг упоминаний в отраслевых СМИ для оценки репутации
## Ограничения
Используются только легальные, открытые, бесплатные источники 
Бесплатный доступ DaData позволяет автоматически получить только по названию компании её ИНН, ОКВЭД и др. реквизиты, адрес. Имеет ограничение в 10 000 запросов в день
Бесплатный доступ Checko ограничен 100 запрасоми в день.
## Роль LLM 
- Обработки текста - извлечение ключевых услуг из описаний компаний, создание кратких выдержек из описаний деятельности
- Генерации запросов - оптимизация поисковых запросов для расширения базы
- Обработка выводов - для простого считывания и дальнейшего слияния разных файлов
LLM ускорил анализ текстовой информации, но финальная верификация данных проводилась вручную.
